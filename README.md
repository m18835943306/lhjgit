# 中心用电项目

## 所需环境及安装依赖注意事项

- Node.js 14+
- vite ^4.3.2
- Antd ^5.4.7
- react ^18.2.0
- react-dom ^18.2.0
- react-router-dom ^6.11.1

### 依赖安装

```bash
pnpm install
```

## package.json scripts

- prepare 安装 husky
- dev 本地开发命令
- build 打包命令
- preview 预览命令！注意先执行 build， 在执行 preview
